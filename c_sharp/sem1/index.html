Programe facute in U.A.V.

Programele au fost testate in Microsoft Visual C# 2008 Express Edition, program ce poate fi descarcat gratuit de aici.
Pentru a intelege aceste exemple este recomandat sa aveti notiuni fundamentale de programare.
  1. Hello world program - Primul program in C#
  2. Exemple de variabile - Tipuri de variabile in C# si System.Console vs Console

  7. Numar prim - Sa se stabileasca daca un numar citit e prim sau nu. Se explica aici si ce inseamna n%d
  8. Descompunere in factori primi - Sa se descompuna un numar citit in factori primi. Ex: 12 = 22 * 31
  9. Min si max dintr-un vector - Sa se afle max si min dintr-un vector de nre intregi. Adica cel mai mare numar si cel mai mic.
  10. Proprietate privata a clasei - Se face o clasa Test care are o proprietate privata int x si se face o functie publica pentru setarea unei valori in x si una care returneaza valoarea lui x.

In caz contrar linile i raman neschimbate. Sa se afiseze matricele obinute. (Neclara: Primele doua elemente ale linilor sunt egale adica a[i,1]=b[i,1] cele 2 elemente fiind primul din a si primul din b sau primele 2 din a sa fie egale cu primele 2 din b? rezolvat pe a doua metoda)

Sa se elimine elementul de valoare maxima daca se apasa litera M. Respectiv valoarea minima daca se apasa m. Afisand lista ramasa. Procedeul se incheie la introducerea altui caracter in afara celor 2 sau la epuizarea listei.
  45. Eliminare de pe pozitii prime - Fiind data o lista liniara sa se elimine elementele de pe pozitiile numere prime. Exemplu: (2, 9, 13, 5, 67, 11) -> (2, 5, 11)
  46. Algoritm pe vector - Se da o lista de n numere. Sa se adauge dupa fiecare element de pe pozitiile impare P un numar de P elemente avand ca si continut valoarea elementului de pe pozitia P. Exemplu: (2, 4, 6, 11) -> (2, 2, 4, 6, 6, 6, 6, 11)
  47. Algoritm pe vector - Se da n numere memorate intr-o lista. Sa se elimine elementele Xi pentru care suma elementelor de la Xi la X(i-1) e egala cu Xi. Sa se elimine apoi din lista elementele negative. Exemplu: (1, 2, 3, -4, 5) -> (1, 2, 5)

  53. String.StartsWith() - s1.StartsWith(s2) returneaza true daca s1 incepe cu sirul de caractere s2 si false altfel.
  54. String.Insert() - s1.Insert(3, s2) returneaza noul String obtinut prin introducerea lui s2 in s1 la pozitia 3.
  55. String.Replace() - s.Replace(s1, s2) returneaza noul String obtinut din stringul s prin inlocuirea tuturor aparitilor lui s1 cu s2.
  56. String.Trim() - Trim() sterge caracterele date ca parametru de la inceputul si sfarsitul unui String.
  57. Sufixele unui cuvant - Se citeste un cuvant. Sa se afiseze toate sufixele sale. Exemplu: "azi" -> "a", "az", "azi"
  58. Eliminarea pe rand a unui caracter - Se citeste un cuvant. Sa se afiseze toate cuvintele obtinute prin eliminarea pe rand a unui caracter. Exemplu: "abc" -> "bc", "ac", "ab"

  60. Cuvinte separate - Se da un text in care cuvintele sunt separate prin un singur spatiu, virgula sau punct. Sa se afiseze cuvintele separate pe cate un rand.
  61. Formarea unui numar palindrom - Se citeste un numar n natural din 8 cifre. Sa se construiasca cu ajutorul cifrelor sale cel mai mare nr palindrom posibil. Daca nu e posibil se va afisa 0. Ex n=21552155 => 55211255

Algoritm pe vector - Dandu-se o secventa de numere reale sa se gaseasca o subsecventa de numere consecutive cu proprietatea ca suma elementelor e maxima in raport cu toate subsecventele de elemente consecutive. Altfel spus Fie un vector cu n elemente : sa se gaseasca 2 indici i si j astfel incat suma elementelor dintre i si j(inclusiv se trece in suma elementele de la pozitia i si pozitia j) ... astfel incat suma sa fie cat se poate de mare. Exemplu (1, 2, 3, 4, 5, 6, 7, 8, -9, 1) -> (1, 2, 3, 4, 5, 6, 7, 8) si (1, 4, 1, 5, 7, -10, 4) -> (1, 4, 1, 5, 7)
